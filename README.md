# CraftComm: チャットコミュニケーションの再発明

Minecraft: Java Edition向けサーバプラグイン「CraftComm」のリポジトリです。

本プラグインは、Minecraftのサーバで行われるチャットの動作を根本から変更し、チャットが届く範囲に距離の概念をもたらします。

## 特徴

- Minecraft 1.16以降で実装されている、チャットのミュート機能に対応
- サーバ側でチャットの内容が書き換わった際に、クライアントに通知する機能に対応
- チャットの発言内容が届く範囲を半径128ブロック以内に制限
- 発言内容の取りこぼしが周囲で発生しにくいように、発言が届いたプレイヤーからさらに発言が伝播するように動作する
